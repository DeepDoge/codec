# Migration Guide

Breaking changes introduced in `@nomadshiba/codec` **0.2.0**.

---

# OptionCodec Renamed to NullableCodec

`OptionCodec` has been renamed to `NullableCodec` to better reflect its
semantics (nullable values, i.e. `T | null`).

The associated type helpers have been renamed accordingly.

| Before           | After              |
| ---------------- | ------------------ |
| `OptionCodec<T>` | `NullableCodec<T>` |
| `OptionGeneric`  | `NullableGeneric`  |
| `OptionValue<T>` | `NullableValue<T>` |

Example:

```ts
// Before
import { OptionCodec, U32 } from "@nomadshiba/codec";
const maybeU32 = new OptionCodec(U32);

// After
import { NullableCodec, U32 } from "@nomadshiba/codec";
const maybeU32 = new NullableCodec(U32);
```

---

# New: OptionalCodec

A new `OptionalCodec` is available for values that may be `undefined` instead of
`null` (`T | undefined`).

Wire format is identical to `NullableCodec` (`0x00` = absent, `0x01` + payload =
present), but decodes to `undefined` rather than `null`.

```ts
import { OptionalCodec, U32 } from "@nomadshiba/codec";

const maybeU32 = new OptionalCodec(U32);
maybeU32.encode(undefined); // [0x00]
maybeU32.encode(42); // [0x01, 0x00, 0x00, 0x00, 0x2a]
```

---

# EnumCodec Renamed to UnionCodec

`EnumCodec` has been renamed to `UnionCodec`. The name better reflects the
concept — TypeScript already has its own `enum` keyword with different
semantics, and modern languages (TypeScript, Zig, Gleam) use "union" for tagged
unions.

The associated type helpers have been renamed accordingly.

| Before         | After           |
| -------------- | --------------- |
| `EnumCodec<T>` | `UnionCodec<T>` |
| `EnumGeneric`  | `UnionGeneric`  |
| `EnumValue<T>` | `UnionValue<T>` |
| `EnumOptions`  | `UnionOptions`  |

Example:

```ts
// Before
import { EnumCodec, StringCodec, U8 } from "@nomadshiba/codec";
const Event = new EnumCodec({ Click: U8, Message: new StringCodec() });

// After
import { StringCodec, U8, UnionCodec } from "@nomadshiba/codec";
const Event = new UnionCodec({ Click: U8, Message: new StringCodec() });
```
